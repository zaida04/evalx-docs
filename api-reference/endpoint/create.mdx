---
title: "Run Code"
api: "POST https://api.evalx.sh/eval"
description: "This endpoint runs code and returns the result"
---

### Body

<ParamField body="code" type="string">
  This is the code that will be run. It can be any valid code in the language you specify.
</ParamField>

<ParamField body="language" type="string">
  This is the language that your code belongs to.
</ParamField>

### Response

<ResponseField name="success" type="number">
  Indicates whether the call was successful. 1 if successful, 0 if not.
</ResponseField>

<ResponseField name="user_group" type="object">

The contents of the user group

</ResponseField>

<RequestExample>

```bash Example Request
curl -X POST \
    -H "Content-Type: application/json" \
    -H "Authorization: test-key-1" \
    -d '{"code": "console.log(\'Hello world!\')", "language": "js"}' \
https://evalx.sh/api/eval
  
```

</RequestExample>

<ResponseExample>

```json Response
{ "output": "Hello world!" }
```

</ResponseExample>
